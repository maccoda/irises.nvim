# irises.nvim

A theme inspired by Irises, the plant and Van Gogh's painting.

Supports tree sitter and semantic highlighting.

## Installing

### Lazy

```lua
{
  "maccoda/irises.nvim",
  lazy = false,
  priority = 1000,
  opts = {},
}
```

## Usage

```vim
colorscheme irises
```

```lua
vim.cmd([[colorscheme irises]])
```
